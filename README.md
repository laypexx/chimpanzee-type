# 🐒 Chimpanzee Type

A minimalist, browser-based **code typing speed test** — practice typing real code snippets and measure your speed and accuracy as a developer.

## 🚀 Live Demo

👉 [laypexx.github.io/chimpanzee-type](https://laypexx.github.io/chimpanzee-type/)

## ✨ Features

- Code-focused typing test with real code snippets
- WPM (words per minute) and accuracy tracking
- Syntax-aware input — practice typing actual code, not just random words
- Clean, distraction-free interface
- Runs entirely in the browser — no install, no login

## 🛠️ Tech Stack

- HTML, CSS, JavaScript
- Hosted via GitHub Pages

## 📦 Getting Started

Clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/laypexx/chimpanzee-type.git
cd chimpanzee-type
open index.html
```

No build step or dependencies required.
